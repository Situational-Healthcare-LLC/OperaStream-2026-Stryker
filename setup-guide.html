🚀 OperaStream Setup

Get your MVP running in 3 minutes

📋 Current Status

Your Supabase project is connected. You need to create the database tables and add sample data. This takes about 3 minutes.

1

Create Database Tables

Click this button to open your Supabase SQL Editor:

Open SQL Editor →

Then:

  • You'll see a blank SQL editor
  • Open the migration.sql file from this project
  • Copy ALL the content (Ctrl+A, Ctrl+C)
  • Paste into the editor (Ctrl+V)
  • Click RUN button (or press Ctrl+Enter)
  • Wait for "Success" message (~5 seconds)
2

Add Sample Data

After tables are created, run this in your terminal:

node scripts/direct-seed.mjs

This adds:

  • 10 sample patients with medical records
  • Lab results and vital signs
  • Healthcare team members
  • External contacts database
3

Create Storage Buckets

For the Presentation Manager feature, create 4 storage buckets:

Open Storage Settings →

Click "Create new bucket" and add these buckets (ALL must be PUBLIC):

  • presentation-images - Public: YES
  • presentation-videos - Public: YES
  • presentation-pacs - Public: YES
  • presentation-documents - Public: YES

🎉 Once complete, your MVP is ready for presentations!
